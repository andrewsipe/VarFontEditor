VarFont Studio (alpha) — install

1. Drag VarFontStudio.app into Applications.
2. Run “Allow First Launch” (same folder as this file):
   • Preferred: double-click it.
   • If macOS blocks that: Control-click → Open.
   • Or drag “Allow First Launch.command” into a Terminal
     window and press Return — the script prints what it does.

That only removes the download quarantine so Gatekeeper will
open this ad-hoc alpha. It does not change your fonts.

Why? Alpha builds are not Apple-notarized. Recent macOS often
shows only “Done” / “Move to Trash” on double-click — no
“Open Anyway” in that dialog.

Alternate (no helper): System Settings → Privacy & Security →
Open Anyway (after clicking Done on the block alert), or:

  xattr -cr /Applications/VarFontStudio.app
  open /Applications/VarFontStudio.app
