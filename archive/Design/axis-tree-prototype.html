Axis tree layout prototype

This is a static HTML/CSS stand-in for AxisTreePanel / AxisTreeAxisHeader / AxisStopTableHeader / AxisTreeStopRow, built from the exact spacing constants in AxisBlockLayout and the shared tokens in StudioDesign.swift. Every CSS custom property in :root is commented with the Swift constant it mirrors — tune a value here, then copy the number back into AxisTreeModels.swift / StudioDesign.swift.

The window in the mock titlebar is resizable (drag its bottom-right corner) — use it the same way you'd measure the real app: the "divider-to-divider" width you'd read in the running app is the content width figure in the readout above the window, not the outer window width (which also includes the left/right scroll insets).

Rows are interactive: click a stop row to select it (tests the highlight vs. the dashed buttons below it), click an Elided radio to move the elidable stop within that axis, click a header row to expand/collapse, click a switch to toggle it. Colors are close approximations of the system dark-appearance colors used in SwiftUI (Color.secondary, accentColor, etc.) — they're here for context, not pixel-exact proofing. The layout/spacing is the part worth trusting.

Outer window: –px Table content width (divider-to-divider): –px — drag the window's bottom-right corner to resize
